Χρόνια μας Πολλά, Βενβάνο μου ❤️

8 Χρόνια με άπειρη αγάπη και πάρα πολυ δυνατά συναισθήματα. Σε ευχαριστώ για όλα αυτά τα συναισθήματα και τις στιγμές που μου έχεις χαρίσει. Θα συνεχίσω να προσπαθώ καθημερινά να σε κάνω ευτυχισμένη και να βλέπω το πιο όμορφο χαμόγελο που έχει υπάρξει ποτέ. Όπως λεει και το τραγούδι όλα στην ζωή είναι γραμμένα.... Σε Βενβάνω για πάντα.
